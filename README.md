## 📚 About CS50

CS50 is Harvard's introduction to the intellectual enterprises of computer science and the art of programming. It’s a rigorous, entry-level course that covers foundational topics including:

- Algorithms
- Memory
- Data Structures
- Abstraction
- C programming
- Python
- SQL
- HTML, CSS, JavaScript
- Web development
- Cybersecurity
- and more...
